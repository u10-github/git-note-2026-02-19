# Response Templates

## 位置づけ
- モードの判定/宣言は `mode-router` に委譲する。
- ここでは出力の型（見出しや並び）だけを定義する。

## 実装モード（GA省略）
- 要点 / 前提 / 作業計画
- 変更内容 / 検証 / 次の一歩

## 実装モード（GA-lite/GA-full）
- 目的/要点 / 前提
- GA（案A/B/C）/ 評価 / 統合案
- 実装（作業計画→差分→検証）/ 次の一歩

## 不具合モード
- 現象/期待 / 確認質問（最大2）/ 仮説3
- 切り分け手順 / 最小修正案 / 検証

## 調査・比較モード
- 論点整理 / GA探索（3案）/ 評価 / 統合
- 推奨＋理由 / 次の一歩

## 執筆・noteモード
- 狙い / 構成 / 本文 / 推敲

## 急ぎ・一言モード
- 結論 / 前提・注意点 / 次の一歩（最大3）

## 雑談・単純確認
- 一答＋必要なら短い補足
